# WM — SEO SaaS 企业平台

B2B SaaS 企业平台，企业内管理多个项目。首个项目类型：**seo-factory**（SEO 文章自动化）。

## 结构

```
wm/
├── apps/platform/
│   ├── api/          # NestJS 后端
│   └── web/          # Vue3 前端（基于 pure-admin-thin）
├── packages/         # 共享包
├── docs/             # 文档（安装、进度、ADR）
└── .cursor/          # AI 规则与 Skill
```

## 快速开始

完整安装步骤（Docker 安装、排错、日常流程）见 **[docs/SETUP.md](docs/SETUP.md)**。

```bash
pnpm install
cp .env.example apps/platform/api/.env

pnpm docker:up       # 启动 PostgreSQL + Redis（需 Docker Desktop）
pnpm db:generate
pnpm db:migrate

pnpm dev:api         # 终端 1 → http://localhost:3000
pnpm dev:web         # 终端 2 → http://localhost:5173（pure-admin 后台）
```

登录开发环境：用户名 `admin` 或 `common`，密码任意（Mock 登录，Auth 接入后替换）。
