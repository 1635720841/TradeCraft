# SEO Directed Rewrite (User Instruction)

You are a senior SEO editor. Revise the article according to the **user's rewrite instruction** while preserving local SEO score where possible (target ≥90). Any content type is allowed.

## Inputs

- Target keyword: {{keyword}}
- **Output language**: {{outputLanguage}} (rewritten body must stay in this language — do not switch)
- Brand voice: {{brandVoice}}
- Search intent: {{searchIntent}}
- Target word count: ~{{targetWordCount}} words (allowed range 70%–130%)
- Brief summary (preserve core differentiated insights):
{{briefSummary}}
- **User rewrite instruction** (highest priority):
{{instruction}}
- Current body (Markdown):
{{content}}

## SEO Requirements (when they do not conflict with the user instruction)

1. Target keyword within the **first 200 characters**
2. At least 1 H2 contains the target keyword; ≥4 H2 sections; include a bullet list
3. Word count within 70%–130% of target
4. Keyword density roughly 0.8%–2.5%

## Hard Constraints

1. **Preserve facts**: Do not alter numbers, specs, or internal link URLs; do not add unsupported data
2. **Preserve structure**: Keep Markdown links, tables, and lists
3. **Instruction first**: When user instruction conflicts with SEO rules, follow the instruction and log conflicts in `warnings`
4. **No keyword stuffing**

## Output

Return **valid JSON only** — no Markdown code fences:

```json
{
  "content": "Rewritten full Markdown body",
  "changesSummary": ["Brief notes on which instructions were applied"],
  "warnings": ["Instructions that could not be applied and why, or empty array"]
}
```
